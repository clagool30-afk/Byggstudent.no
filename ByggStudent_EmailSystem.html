Steg-for-steg oppsett med EmailJS
⚙️
Oppsett — 15 minutter
1
Opprett gratis konto på EmailJS
Gå til emailjs.com → Sign Up → velg Gmail som e-posttjeneste og koble til din Outlook/Hotmail
2
Lag to e-postmaler
Én mal for studenten (velkomst-e-post) og én for deg (varsel om ny registrering). Bruk variablene {{navn}}, {{plan}}
3
Kopier API-nøklene dine
Fra EmailJS dashboard: kopier Public Key, Service ID og Template ID
emailjs.init("DIN_PUBLIC_KEY")
4
Lim inn koden i nettsiden din
Legg til script-taggen i HTML og kall funksjonen når skjemaet sendes. Se kode-eksempelet til høyre →
5
Test med din egen e-post
Fyll ut skjemaet selv og sjekk at du mottar begge e-postene. Ferdig!
💻
Kode-eksempel
// 1. Legg til i <head>
<script src="https://cdn.emailjs.com/
  dist/email.min.js"></script>

// 2. Initialiser
emailjs.init("DIN_PUBLIC_KEY");

// 3. Send e-poster ved registrering
function sendEmails(navn, epost, plan) {

  // E-post til studenten
  emailjs.send(
    "SERVICE_ID",
    "TEMPLATE_STUDENT",
    { navn, epost, plan,
      betalingslenke: getPayLink(plan) }
  );

  // Varsel til deg (Claudio)
  emailjs.send(
    "SERVICE_ID",
    "TEMPLATE_ADMIN",
    { navn, epost, plan,
      tid: new Date().toLocaleString("no-NO") }
  );
}

// 4. Kall funksjonen fra skjemaet
function sendRegistration() {
  const navn = getElementById("reg-name").value;
  const epost = getElementById("reg-email").value;
  const plan = getElementById("reg-plan").value;
  sendEmails(navn, epost, plan);
}
✅ Ingen server nødvendig — alt kjører direkte fra HTML-filen. Du trenger ikke programmering-kunnskap for å sette dette opp.
