Last opp filene til GitHub
GitHub er der filene dine lagres — som en sky-mappe for kode. Gratis og tar 3 minutter.
Enkelt
Opprett GitHub-konto og nytt prosjekt
~2 min
1
Gå til github.com og klikk "Sign up"
Bruk din Outlook-e-post. Velg gratis plan (Free).
2
Klikk "New repository" (grønn knapp)
Gi det navnet byggstudent-no. Sett til Public. Klikk "Create repository".
3
Last opp filene dine
Klikk "uploading an existing file" → dra alle 4 HTML-filene inn → klikk "Commit changes".
✓ index.html · admin.html · dashboard.html · email-system.html
Du trenger ikke installere noe. Alt skjer i nettleseren på github.com
Tips
Filstruktur i GitHub
📁 byggstudent-no/
  📄 index.html    ← Forsiden
  📄 dashboard.html  ← Studentportal
  📄 admin.html    ← Admin-panel
  📁 pdf/         ← (valgfri lokal mappe)
  📄 README.md    ← Beskrivelse
Ikke last opp PDF-filer til GitHub — bruk Google Drive til det. GitHub er kun for HTML/CSS/JS-filer.
2
Publiser på Vercel — gratis hosting
Vercel kobler seg til GitHub og publiserer nettsiden din automatisk. Hver gang du oppdaterer en fil på GitHub, oppdateres nettsiden automatisk.
Enkelt
Deploy til Vercel på 3 minutter
~3 min
1
Gå til vercel.com og klikk "Sign Up"
Velg "Continue with GitHub" — det kobler kontoene automatisk.
2
Klikk "Add New → Project"
Du ser GitHub-repoene dine. Klikk "Import" ved siden av byggstudent-no.
3
Klikk "Deploy" — ikke endre noe
Vercel oppdager filene automatisk. La alle innstillinger stå og klikk Deploy.

4
Aktiver automatiske e-poster
Koble EmailJS til nettsiden så du mottar varsler og studenten får velkomst-e-post automatisk.
Enkelt
EmailJS oppsett — 5 minutter
~5 min
1
Opprett konto på emailjs.com
Gratis — logg inn med Google eller e-post
2
Email Services → Add New Service → Gmail/Outlook
Koble til din Outlook-konto. Gi tjenesten navn: byggstudent
3
Email Templates → Create New Template
Lag to maler: template_student (velkomst) og template_admin (varsel til deg). Kopier teksten fra e-postsystem-siden.
4
Kopier nøklene dine
Account → API Keys → kopier Public Key. Finn også Service ID og Template IDs fra dashbordet.
5
Oppdater index.html med nøklene dine
Finn linjen med DIN_PUBLIC_KEY i koden og bytt ut med din faktiske nøkkel. Last opp til GitHub → automatisk oppdatering!
// Lim inn i <head> på index.html <script src="https://cdn.emailjs.com/ dist/email.min.js"></script> <script> emailjs.init("abc123XYZ"); // ↑ Din Public Key fra EmailJS </script> // I sendRegistration() funksjonen: emailjs.send( "service_byggstudent", // Service ID "template_student", // Til student { navn, epost, plan } ); emailjs.send( "service_byggstudent", "template_admin", // Til deg { navn, epost, plan, tid } );
Ferdig! Test ved å registrere deg selv med din egen e-post.
